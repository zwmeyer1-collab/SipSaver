# SipSaver Roadmap

## Core Idea

SipSaver helps people find the best happy hour deals, nearby events, and go-out options in one place.

## Original Idea List, Cleaned Up

- Scrape or collect happy hour deals in a local area
- Let bars submit or verify official deals
- Offer exclusive app-only or member-only deals
- Add a social planning feature called "Hoppy Hour"
- Let users create a night plan, invite friends, and vote on bars
- Show where places are active or "popping"
- Add rewards for check-ins, deal usage, and inviting friends
- Support points that can be redeemed for discounts or perks
- Include a real-time vibe indicator
- Build a map view
- Show live music, DJs, and game nights
- Add a "Bar Crawl Mode" that generates a route based on distance and time windows

## Recommended MVP

Build the first version as a website focused on utility.

### MVP Features

- Area-based happy hour listings
- Search and filtering by deal type or event type
- Simple map-based browsing
- Deal detail cards with times and highlights
- Pages for events like live music, DJs, and trivia
- A bar claim form for business owners

## Why Website First

- Faster to launch and easier to update
- Better for search engine discovery
- Easier to validate whether people want deal discovery before building social features
- Lets the product earn trust before asking users to create accounts

## Recommended Build Order

1. Discovery website
2. Verified business submissions
3. User accounts, favorites, and saved plans
4. Social planning and friend activity
5. Native app experience

## Product Risks To Think About

- Scraped deal data can become outdated fast
- Business verification will matter if the app is going to be trusted
- "Real-time vibe" features are hard without a lot of active users
- Rewards and loyalty systems add complexity early

## Best Positioning For Launch

Find the best happy hour deals near you, live and in one place.
